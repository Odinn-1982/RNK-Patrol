# Bark Audio Files

Place your guard voice clips in subdirectories by category:

- `spawn/` - Sounds when guards appear
- `despawn/` - Sounds when guards disappear  
- `capture/` - Sounds when capturing a player
- `disregard/` - Sounds when guard ignores player
- `theft/` - Sounds when stealing from player
- `bribery_accept/` - Sounds when accepting a bribe
- `bribery_reject/` - Sounds when rejecting a bribe
- `bribery_doublecross/` - Sounds when taking bribe AND arresting

## Supported Formats
- MP3
- OGG
- WAV
- WEBM

## Naming Convention
Multiple files in a category will be randomly selected.
Example: `capture/halt_01.mp3`, `capture/halt_02.mp3`, `capture/gotcha.mp3`
